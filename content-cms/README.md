# CMS Collection Tool

A headless Content Management System that allows you to define custom schemas and manage content collections through a simple dashboard and public APIs.

## What This Does

This CMS lets you:
- **Define schemas** (like "Blog Post" with fields: title, content, author, publish date, image, SEO metadata)
- **Create collections** (instances of your schemas with actual content)
- **Access content via API** (use API keys to fetch content for your websites/apps)

Think of it like a database table designer where:
- **Schema** = Class definition (structure)
- **Collection** = Instance of that class (actual data)

## Use Cases

- **Blogs** - Define a blog schema, create posts, fetch via API
- **E-commerce stores** - Product catalogs, categories
- **Marketing websites** - Landing pages, testimonials, case studies
- Similar to Webflow CMS but headless (API-first)
